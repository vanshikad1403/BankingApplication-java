# Banking Application in Java  
A console-based Java application that simulates basic banking operations such as creating an account, depositing money, withdrawing funds, and updating contact details.

## Features  
- Create new bank accounts with auto-generated account numbers  
- Deposit and withdraw money  
- Validate balance before withdrawal  
- View complete account details  
- Update email and phone number  
- Store multiple accounts using arrays  

## Concepts Used  
- Object-Oriented Programming (OOP)  
- Classes and objects  
- Encapsulation with private fields and public methods  
- Input handling using Scanner  
- Basic data validation  
- Menu-driven console interface  

## How It Works  
The application stores multiple account objects in an array and allows users to perform operations by selecting options from a menu. Each account maintains its own details and provides methods for deposit, withdrawal, and updating contact information.

## Technologies  
- Java  
- Scanner API  
- OOP Principles
